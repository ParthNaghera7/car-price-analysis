# car-price-analysis
🚗 **Car Price Analysis** is a data analytics project focused on understanding used car auction prices. The project involves cleaning and preprocessing raw vehicle data, handling missing values, validating VIN numbers, standardizing categorical features, and removing outliers using statistical methods. After data preparation, Exploratory Data Analysis (EDA) is performed to uncover trends and relationships between factors such as vehicle age, mileage, transmission type, MMR value, and selling price. The analysis reveals key insights that can help understand the used car market and serves as a foundation for future machine learning models for car price prediction. The project is developed using Python with libraries including Pandas, NumPy, Matplotlib, and Seaborn.
